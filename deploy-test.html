🚀 BudgetWise Cloud Deployment

⚠️ Note: These are temporary deployment options. For permanent hosting, consider paid cloud services.

🌐 Option 1: Netlify Drop (Easiest - No Account)

Deploy frontend instantly by dragging and dropping files.

  1. Open Netlify Drop
  2. Drag the deploy/frontend folder to the page
  3. Get your live URL instantly!

Backend: Use ngrok (see Option 3)

⚡ Option 2: Surge.sh (Command Line)

Deploy frontend from command line, get a free .surge.sh domain.

npm install -g surge
cd deploy/frontend
surge --domain budgetwise-random.surge.sh

Backend: Use ngrok or localtunnel

🌐 Option 3: ngrok (Backend Public Access)

Make your local backend accessible from anywhere temporarily.

npm install -g ngrok
cd backend
mvn spring-boot:run
// In another terminal:
ngrok http 8082

Get a temporary URL like: https://abc123.ngrok.io

🔧 Option 4: localtunnel (Alternative to ngrok)

Similar to ngrok but uses a different service.

npm install -g localtunnel
lt --port 8082

🐳 Option 5: Play-with-Docker (Full Stack)

Test your containerized app in a free Docker environment.

  1. Go to Play-with-Docker
  2. Create a new instance
  3. Upload your docker-compose.yml
  4. Run: docker-compose --profile cloud up

📱 Option 6: Mobile Testing

Test your deployed app on mobile devices.

Use services like:

  • BrowserStack (Free tier)
  • LambdaTest (Free tier)
